Machine learning Assignment-1¶

Problem Statement¶

The Market historical dataset of real estate valuation are collected From sindian Dist.New Taipei city,Taiwan. The real estate valuation is a regression problem.The dataset was randomly split into training data set(2/3samples) and the testing data set(1/3 samples).

we have to predict the house price of unit area using some given information house age,transaction date,distance to the nearest MRT station,number of convenience stores,the geographic coordinate latitude and longitude.

we have given a csv file containing historical data of houses
Dataset source : http://archive.ics.uci.edu/ml/machine-learning-databases/00477/

If we can do so for the historical data,then we should able to estimate the house prices for new houses too.
